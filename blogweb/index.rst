Code Sprints
------------

**Q:** *What is a code sprint?*

A **code sprint** is an accelerated research and development program, where we
pair talented developers with senior researchers and engineers for 3-6 months of extended
programming work. The model is inspired by the Google Summer Of Code
initiative, and is meant to create open source solutions for interesting 2D/3D
computer perception problems. Each code sprint is financially supported by a
different institution (see below).

Code sprints are made possible through the involvement of our host
organization, `Open Perception <http://www.openperception.org>`_. Please see
our `mission page <http://www.openperception.org/mission/>`_ for more
information.

**Q:** *What does this blog represent?*

The Point Cloud Library (PCL) **developer blog** represents a great way to keep
track of daily updates in PCL sponsored code sprints. Our developers are
writing about their experiences and progress updates while working on exciting
PCL projects.
