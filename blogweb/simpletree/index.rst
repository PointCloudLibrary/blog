SimpleTree Code Sprint
---------------

SimpleTree is a PCL based open source tool for the geometrical modelling of trees in the field of computational forestry.
The tool was developed within the 

Lin2Value project fundet by the German ministry BMBF.

.. image:: images/bmbf.png
   :align: center
   :width: 300

The modelling part was done at the Chair for Forest Growth


.. image:: images/iww.png
   :align: center
   :width: 300

within the German-Sino project Lin2Value


.. image:: images/lin2value.png
   :align: center
   :width: 300

For a complete list of all the present and past PCL code sprints please visit http://www.pointclouds.org/blog.
